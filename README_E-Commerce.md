# European E-Commerce Business Insights Dashboard

## Overview
This project focuses on analyzing e-commerce data across Europe to understand customer behavior, product performance, and regional trends.

## Key Features
- Revenue and profit tracking
- Customer segmentation analysis
- Product performance insights
- Regional sales visualization

## Key Insights
- Key regions contribute the majority of revenue
- A small number of products drive most sales
- Customer behavior patterns indicate strong repeat purchasing trends

## Tools Used
- Power BI
- Excel
